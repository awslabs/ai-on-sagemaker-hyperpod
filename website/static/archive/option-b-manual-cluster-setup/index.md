This is the traditional way of setting up the cluster. If you'd like to do it manually, you may follow the steps below. Please make sure you have:
1. Administrator Access to your console via IAM. Run `aws configure`
2. Completed the [prerequisite steps](/00-setup), and deployed the CloudFormation stack(s).
3. A Linux based development environment (macOS is great!)
